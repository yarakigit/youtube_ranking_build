#  GitHub Pages 自動更新
- [作成したサイト](https://yarakigit.github.io/youtube_ranking_build/)
- GitHub ActionsでGitHub Pagesを自動更新します。
  - **cron**でワークフローが定期実行されます。
- 静的サイトジェネレーターとして[Pelican](https://github.com/getpelican/pelican)を使っています。
    - [Flex](https://github.com/alexandrevicenzi/Flex)テーマを使用
- Google Analyticsを埋め込んでいます。
  -  プライバシーポリシーの明記が必要。
     -  [【コピペ可】Googleアナリティクスの設置で必須なプライバシーポリシーの書き方](https://www.iscle.com/web-it/g-drive/g-analytics-privacy-policy.html)を参考にしました。

- YouTube Data APIを用いてYouTubeのデータを取得しています。
